# 📘 Help

## 🛠️ How to Use
1. This will only work on Windows
2. Close Roblox before running the FPS Changer.
3. If Roblox is open, choose **Yes** to close it or **Cancel** to stop.
4. Enter the FPS you want.
5. Setting the FPS to **0** will make it **uncapped**.
6. After the FPS is changed, reopen Roblox.

## ⚙️ How It Works
The tool updates Roblox’s client settings file and changes the **FramerateCap** value.  
This is a normal client‑side setting and does **not** modify gameplay, memory, or anything unsafe.  
It is allowed by Roblox TOS because it only adjusts a configuration value Roblox already uses.
